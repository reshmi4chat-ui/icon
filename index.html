This app is prepared as a single-screen Android calculator experience. These are manual release steps only, not automated publishing.

Release prep
  • Create the Android package from this hosted app so launch opens directly to Pulse Calculator.
  • Use the Pulse Calculator name and matching icon for package, listing, and installed app identity.
  • Upload the generated Android package to Google Play Console, then add screenshots, app details, and store policy answers there.
